# VK Properties Display

Modern real-estate showcase website built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Local development

```sh
npm install
npm run dev
```

## Production build

```sh
npm run build
npm run preview
```

## Deploy on Vercel

1. Push this repository to GitHub.
2. Import the repo in Vercel.
3. Use these settings (Vercel usually auto-detects them):
   - Framework Preset: `Vite`
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. Deploy.

## Stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
